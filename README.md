## 项目定位

本项目系统整理 OUC-HWU 计科中外合作办学项目中的培养规则、课程学习、语言要求、科研训练、升学申请、赴英准备与留学生活经验。文档以学生视角记录长期可复用的信息，目标是降低信息差，并为后续年级提供可检索、可修订、可持续维护的公共参考。

本指南不是官方政策文件。涉及培养方案、语言要求、签证、住宿、申请规则和学校流程时，请以当年中国海洋大学、赫瑞瓦特大学及目标院校官方通知为准。

## 参与方式

欢迎补充课程经验、专题文章、申请时间线、生活攻略、资源链接和勘误。建议流程如下：

1. 在 [Issues](https://github.com/zhaorui-bi/OUC-HW-Survial-Guidance/issues) 提出问题或修改建议。
2. Fork 仓库后修改 `Survial_Guide/docs/` 下的 Markdown 文档。
3. 提交 Pull Request，并说明修改动机、适用年级、信息来源和可能的时效性限制。
