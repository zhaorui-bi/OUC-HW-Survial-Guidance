## 如何使用

建议优先阅读在线文档：

[https://ouc-hw-survial-guidance.readthedocs.io/zh-cn/latest/](https://ouc-hw-survial-guidance.readthedocs.io/zh-cn/latest/)

你可以按自己的阶段选择专题：

| 场景 | 推荐阅读 |
| --- | --- |
| 刚进入项目，想了解培养结构 | 首页、项目基础、课程与成绩相关内容 |
| 准备推免或海外申请 | 保研规划、申研规划、本科生如何做科研 |
| 准备赴英学习与生活 | 雅思备考、行前准备、留学生做饭菜谱 |
| 想探索方向或补充技能 | AI4Science 学习规划、机器人学习规划、计算物理学、Verilog 使用说明 |

本文档是学生经验与公开资料的整理，不替代学校、学院、签证中心或目标院校的正式通知。涉及培养方案、语言要求、申请规则、签证和住宿等关键事项时，请以当年官方通知为准。

## 如何维护

为了避免内容重复，本仓库只保留一个主要维护入口：

**主要内容统一维护在 `Survial_Guide/docs/` 目录中。**

维护时请遵循这几个原则：

1. 修改正文时，优先修改对应专题页面，不在仓库根目录新增重复 Markdown。
2. 新增专题时，请同时补充标题、适用对象、更新时间和信息来源。
3. 涉及政策、申请要求、课程安排或费用的信息，请标明适用年份，并提醒读者二次核对官方通知。
4. 投稿或补充案例时，请避免公开个人隐私、未授权聊天记录、申请材料和敏感身份信息。
5. 发现错误、链接失效或内容过期时，建议通过 Issues 提交勘误。

`README.md` 只作为 GitHub 仓库入口页，不承载完整正文；Read the Docs 页面才是正式阅读版本。

## 参与贡献

欢迎补充课程经验、申请时间线、科研训练、赴英准备、生活攻略和资源链接。建议优先通过以下方式参与：

- 在 Issues 中提出问题、勘误或补充建议。
- 直接修改 `Survial_Guide/docs/` 中的对应专题并提交 Pull Request。
- 如果贡献的是个人经验，请尽量说明适用届别、背景条件和可能失效的时间点。
