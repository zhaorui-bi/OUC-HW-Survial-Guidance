## 英语篇 { #英语篇 }

Heriot-Watt University 入学英语要求如下表第三列，总分和小分都必须达标。

<img src="fig/1.jpg" alt="Heriot-Watt University language requirement" class="guide-image guide-image--medium" />

一般情况下建议考 IELTS Academic。学术英语课也会围绕雅思相关内容展开。如果有强烈赴美深造意愿，也可以选择 TOEFL iBT，但一定要注意小分要求。

为什么英语能力重要？

- 赫瑞瓦特大学要求达到 IELTS 6.0（各单项 5.5）或其他同等语言成绩。不达标虽然有语言班，但费用很高。
- 出国申请中，亮眼的雅思或托福成绩是必备项，也是加分项。
- 国内保研和出国申请面试都可能是英文面试。
- CS 和 AI 前沿内容主要以英文论文、课程和文档呈现。
- 大四、硕士、博士阶段的学习、沟通和学术交流都需要英语能力支撑。

### 刷分问题

赫瑞瓦特大学成绩单（英方成绩单）不支持刷分，即第一次考试成绩为最终成绩单分数。

英方课程：[F2R8-ROB curriculum](https://curriculum.hw.ac.uk/programmedetails/F2R8-ROB?termcode=202425)

中国海洋大学成绩单（中方成绩单）支持刷分，同名课程取最高。也就是说，英方课程如果刷分，英方成绩单不更新，但中方成绩单会更新。

刷分策略：

- 尽量刷高学分课程。
- 尽可能刷超低分课程。
- 多修高分通识课以抬高均分。

!!! warning "保研特别注意"
    保研成绩计算通常只看第一次初修成绩，刷分对保研排名可能无效。具体以当年学院政策为准。

## 科研篇 { #科研篇 }

科研训练具有较强的目标差异性，但越早理解其与升学目标之间的关系，越有利于形成稳定的长期规划。

| 目标 | 科研权重 | 建议 |
| --- | --- | --- |
| 海博 | 权重较高 | 建议尽早开始，并形成明确产出 |
| 保研 | 重要补充到关键加分 | 高竞争度院校建议具备科研产出 |
| 海硕 | 视目标学校而定 | 大多数情况下绩点仍是主线 |

备注：计科中外保研率根据 2020 级情况大概在 20% 左右。

必修模块 ML（2 选 1）和 DL 模块建议认真学，熟悉 PyTorch 深度学习框架。后续可以根据兴趣选择 CV、NLP、GNN、XAI 等方向。

!!! note "毕业设计提醒"
    前三年即使没有深入学习 AI / ML，大四毕业设计也大概率需要接触相关内容，建议提前建立基础。

HWU 毕业设计项目链接：[Project proposals](https://projects.hw.ac.uk/Proposals/Index)
