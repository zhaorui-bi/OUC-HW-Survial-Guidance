## 2026-07-02

### 站点视觉

- 将 Read the Docs 站点整体调整为白色简约学术风格：白色页面、细边框、低饱和灰蓝辅助色与克制的信息层级。
- 将正文、导航与标题字体统一为 Times New Roman + 楷体回退，兼顾英文数字与中文文本的学术感。
- 重构首页 hero、路线图卡片、专题导航卡片和表格样式，使页面更接近“学术资料库 + 项目说明书”的浏览体验。
- 调整按钮、链接、表格、图片与提示块的视觉规范，降低阴影和色块使用，增强正式感与阅读稳定性。
- 细化侧栏、搜索结果、页脚、移动端抽屉与卡片 hover 状态，保持全站白色系一致。

### 内容架构

- README 改为仓库级项目门面，突出在线阅读、专题导航、参与方式和本地预览。
- Read the Docs 首页改为结构化入口页，新增路线图、速查表、专题导航和更新记录。
- 补充 README 与首页的信息边界、维护原则、贡献规范和非官方说明。
- 更新日志从占位内容扩展为正式维护页，用于记录站点设计、内容更新和修复事项。
- 补充站点描述、搜索配置、导航体验与 favicon 配置。

### 修复与维护

- 修复申研规划页二维码失效问题，将旧 WeChat 临时二维码替换为稳定 Read the Docs 地址：
  [https://ouc-hw-survial-guidance.readthedocs.io/](https://ouc-hw-survial-guidance.readthedocs.io/)
- 同步更新 `sy1.jpg`、`sy2.png` 与独立二维码图片资源。
- 新增 MathJax 配置文件，补齐 `mkdocs.yml` 中原先引用但不存在的本地脚本。
- 使用临时构建目录验证站点，避免将本地构建产物混入源码变更。

### 协议说明

- 明确项目采用 CC BY-NC-SA 4.0 协议。
- 在 README、首页和站点导航中补充“非商业使用”说明。
- 新增使用协议说明页，便于读者理解署名、非商业和相同方式共享要求。
